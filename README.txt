WGCLOUD是一款高性能的运维监控系统，具备分布式、部署操作简单、私有化部署、安全稳定、轻量实用等特点
支持主机各种指标监测（cpu使用率，cpu温度，内存使用率，磁盘容量，磁盘IO，硬盘SMART健康状态，系统负载，连接数量，
网卡流量，硬件系统信息，GPU，防火墙等）。支持监测服务器上的进程应用、文件防篡改、端口、日志、DOCKER容器、K8S、Kafka、Redis、Nginx、数据库、数据表等资源。
支持监测服务接口API、网络设备（如交换机、路由器、打印机）等。自动生成网络拓扑图，大屏可视化，web SSH（堡垒机），
巡检报告，指令下发批量执行，计划任务，资产管理，工作笔记，设备账号管理，主机异常进程监测，告警信息推送（如邮件、钉钉、微信、短信等）
